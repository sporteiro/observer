Upload Media

  Heading: Subir media
  Instructions: Haga click o arrastre los archivos para subirlos. Tamaño máximo: 10GB.
  Select Files Legend: Seleccione archivos
  Select Files: Arrastre o haga click para seleccionar los archivos a subir.
  File Drop Failed: No es posible obtener la información del archivo. Haga click para seleccionar un archivo diferente.
  Upload Status: Cargando: %1%
  ID3 Unavailable: ID3 / Información EXIF No disponible
  Wait For Upload To Save: No es posible guardar, por favor, espere a que la carga haya finalizado.
